# Bountiq Frontend

Next.js 15 application for the Bountiq platform. App Router, TypeScript, TailwindCSS, shadcn/ui, Framer Motion.

## Local development

```bash
npm install
npm run dev
The dev server starts at http://localhost:3000.

Folder map
Path	Purpose
app/	App Router routes and layouts
components/	UI primitives and feature components
hooks/	React hooks
lib/	Initialized clients and shared helpers
services/	Data access for Firestore and GenLayer
types/	Shared TypeScript types
utils/	Pure helpers (validators, formatters)
config/	Static configuration and constants
styles/	Global stylesheet and design tokens
